# 📰 Fake News Detection in Regional Language

A Machine Learning project that detects fake news in Hindi using Natural Language Processing (NLP) and supervised machine learning algorithms.

---

## 📌 Project Overview

This project aims to identify fake news articles written in Hindi by applying text preprocessing, TF-IDF vectorization, and machine learning classification models.

The system classifies news articles as **Real** or **Fake**, helping reduce misinformation with high accuracy.

---

## 🚀 Features

- Hindi Text Processing
- Data Cleaning & Preprocessing
- TF-IDF Vectorization
- Machine Learning Classification
- Human-in-the-Loop Validation
- Model Performance Evaluation
